 Amazon Clone
This project is a static replica of the Amazon homepage, created using HTML and CSS. It aims to mimic the core structure, layout, and styling of the Amazon website, showcasing basic front-end development skills.

Features
Responsive Layout: The clone is designed to be fully responsive, adapting to different screen sizes, from desktop to mobile.

Navigation Bar: A top navigation bar with links to key sections such as Home, Today's Deals, and Orders.

Product Showcase: A section displaying featured products, similar to Amazon’s homepage.

Styling: Custom CSS to match the color scheme and layout of Amazon’s homepage.

Technologies Used
HTML: Used to structure the content and layout of the webpage.

CSS: Applied for styling and creating a visually appealing design.
